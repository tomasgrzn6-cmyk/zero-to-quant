# zero-to-quant

Documenting my self-taught journey from writing my first line of code to building quantitative finance tools — starting summer 2026, ahead of studying Mathematical Engineering at FSI VUT Brno.

## Why this repo exists

Most learning happens privately and gets thrown away. This repo is the opposite — every step, including mistakes, gets committed here in the open.

## What's inside

- **`python/`** — Python programs, starting from fundamentals (input validation, algorithms)
- **`cpp/`** — C++ programs, progressing through a structured level system (variables → functions → pointers → OOP)
- **`notes/`** — math notes and proof work (coming soon)
